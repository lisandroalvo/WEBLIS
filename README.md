# Draggable Glass Morphism Component

A stunning, interactive glass morphism navigation component with drag functionality.

## Features

- Beautiful glass morphism effect with backdrop blur
- Smooth drag-and-drop functionality (mouse & touch support)
- Multiple background options
- Responsive design
- Production-ready code

## Live Demo

View the live demo at: https://lisandroalvo.github.io/WEBLIS/

## Technologies

- Pure HTML, CSS, and JavaScript
- No dependencies
- Glass morphism design
- Unsplash API for backgrounds
